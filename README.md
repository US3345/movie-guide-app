# 🎬 Movie Guide App

A simple and responsive web application that allows users to search for movies and view detailed information such as rating, runtime, actors, release date, plot, and poster using the **OMDb API**.

---

## 🚀 Features

* 🔍 Search movies by title
* 🎥 Displays movie poster
* ⭐ Shows IMDb rating
* 🏷 Genre displayed as tags
* 📅 Shows release date and runtime
* 👨‍🎤 Displays actors
* 📝 Shows movie plot
* ⚡ Uses **async/await** for API requests
* 📱 Responsive design using **Flexbox & Media Queries**

---

## ⚙️ How It Works

1. User enters a movie name in the search box.
2. JavaScript sends a request to the **OMDb API**.
3. The API returns movie data in JSON format.
4. The application dynamically displays:

   * Poster
   * Title
   * Rating
   * Genre
   * Runtime
   * Actors
   * Plot

---

## 📦 API Used

**OMDb API**

```
http://www.omdbapi.com/?apikey=YOUR_API_KEY&t=movie_name
```
